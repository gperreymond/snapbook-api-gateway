# snapbook
## api / gateway

- SNAPBOOK_MICROSERVICE_DATA_URL
- SNAPBOOK_MICROSERVICE_OPENCV_URL
- SNAPBOOK_VOLUMES_APPLICATIONS
- ENABLE_NEWRELIC
- ENABLE_NPM_TEST

"ENABLE_NEWRELIC = 1" via tutum pour activer le monitoring.

"ENABLE_NPM_TEST = 1" via cloud9 pour activer les tests.
